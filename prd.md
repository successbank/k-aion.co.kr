# 건강 1도씨존 홈페이지 PRD (Product Requirements Document)

## 1. 개요

### 1.1 프로젝트 정보
| 항목 | 내용 |
|------|------|
| 프로젝트명 | 건강 1도씨존 공식 홈페이지 |
| 클라이언트 | ㈜케이아이온 |
| 브랜드 슬로건 | "내 몸을 따뜻하게, 건강 1도씨존" / "1도와 함께, 성공적인 미래" |
| 문서 버전 | v1.0 |
| 작성일 | 2026년 1월 2일 |

### 1.2 프로젝트 배경
㈜케이아이온은 건강 의료기기(저주파, 초음파) 및 건강기능식품 판매 프랜차이즈 사업을 운영하고 있습니다. "건강 1도씨존"이라는 브랜드로 전국에 대리점을 모집하고 운영 중이며, 예비 창업자와 일반 고객을 대상으로 한 공식 홈페이지 구축이 필요합니다.

### 1.3 프로젝트 목표
1. 브랜드 인지도 향상 및 신뢰도 구축
2. 예비 창업자(가맹점주) 모집 및 상담 문의 유도
3. 제품 정보 제공 및 고객 문의 채널 확보
4. 기존 가맹점주를 위한 정보 허브 역할

## 2. 타겟 사용자

### 2.1 Primary User (주요 사용자)
| 사용자 유형 | 특성 | 니즈 |
|------------|------|------|
| 예비 창업자 | 40~60대, 소자본 창업 희망자 | 창업 비용, 수익성, 지원 혜택 정보 |
| 건강 관심 소비자 | 40~70대, 건강관리 관심층 | 제품 효능, 매장 위치, 체험 후기 |

### 2.2 Secondary User (보조 사용자)
| 사용자 유형 | 특성 | 니즈 |
|------------|------|------|
| 기존 가맹점주 | 현재 운영 중인 점주 | 교육 자료, 본사 공지, 마케팅 자료 |
| 지사장/영업이사 | 지역 관리자 | 대리점 관리 정보, 실적 현황 |

## 3. 핵심 기능 요구사항

### 3.1 페이지 구조

```
홈페이지
├── 메인 (랜딩 페이지)
├── 회사소개
│   ├── 기업 소개
│   ├── CEO 인사말
│   └── 연혁/비전
├── 브랜드 소개
│   ├── 건강 1도씨존 소개
│   └── 매장 이미지 갤러리
├── 제품 소개
│   ├── 저주파 치료기
│   ├── 초음파 치료기
│   ├── 통증패치
│   └── 프리미엄 젤
├── 창업 안내 ⭐
│   ├── 사업 모델 소개
│   ├── 창업 비용 및 혜택
│   ├── 계약 조건
│   ├── 지원 프로그램
│   ├── 창업 절차 가이드
│   └── 창업 상담 신청
├── 매장 찾기
│   └── 전국 매장 지도
├── 고객센터
│   ├── 공지사항
│   ├── FAQ
│   ├── 1:1 문의
│   └── 체험 후기
└── 점주 전용 (로그인 필요)
    ├── 교육 자료실
    ├── 마케팅 자료실
    └── 본사 공지
```

### 3.2 주요 기능 상세

#### 🏠 메인 페이지
| 섹션 | 내용 | 우선순위 |
|------|------|----------|
| 히어로 배너 | 브랜드 슬로건 + 핵심 CTA (창업상담/매장찾기) | 필수 |
| 브랜드 소개 | 건강 1도씨존 핵심 가치 3~4가지 | 필수 |
| 창업 혜택 하이라이트 | 2860만원 창업비용, 1330만원 제품지원 등 | 필수 |
| 성공 사례 | 점주 인터뷰/후기 슬라이더 | 권장 |
| 제품 미리보기 | 주요 제품 4종 썸네일 | 권장 |
| 매장 갤러리 | 실제 매장 사진 3~4장 | 선택 |

#### 🚀 창업 안내 (핵심 페이지)
| 기능 | 상세 내용 | 우선순위 |
|------|-----------|----------|
| 사업 구조도 | 본사 → 영업이사 → 지사장 → 1도씨존 계층 시각화 | 필수 |
| 비용/혜택 비교표 | 창업비용 2860만원 vs 지원혜택 명확히 표시 | 필수 |
| 창업 절차 안내 | 계약 → 신고/허가 → 사업자등록 → 교육 → 오픈 | 필수 |
| 체크리스트 다운로드 | 현장 체크리스트 PDF 제공 | 권장 |
| 상담 신청 폼 | 이름, 연락처, 희망지역, 문의내용 | 필수 |
| 비용 시뮬레이터 | 예상 수익/비용 계산기 (선택) | 선택 |

#### 📋 창업 절차 상세 (사업정책 기반)
```
Step 1: 사업장 계약
  └── 현장 방문 → 계약금 지급 → 제품 검토 → 잔금 지급

Step 2: 신고 및 허가
  ├── 의료기기 판매업 신고 (정부24)
  ├── 건강기능식품 법정 교육 이수
  └── 건강기능식품 영업 신고

Step 3: 사업자 등록
  └── 홈택스에서 신청 (첨부: 임대차계약서, 각종 신고증)

Step 4: 대출 안내 (선택)
  ├── 지역 신용보증재단
  ├── 서민금융진흥원
  └── 소상공인시장진흥공단
```

#### 📍 매장 찾기
- 네이버/카카오 지도 API 연동
- 지역별 필터링 (시/도, 구/군)
- 매장 상세정보: 주소, 연락처, 운영시간

#### 👤 점주 전용 공간
- 로그인/회원가입 (점주 인증)
- 교육 자료 다운로드
- 마케팅 자료 (배너, 팜플렛 등)
- 본사 공지사항

## 4. 디자인 요구사항

### 4.1 브랜드 컬러 (제안)
| 용도 | 컬러 | 의미 |
|------|------|------|
| Primary | #E53935 (레드) 또는 #FF6B35 (오렌지) | 따뜻함, 건강, 활력 |
| Secondary | #1976D2 (블루) | 신뢰, 전문성 |
| Accent | #FFC107 (골드/옐로우) | 프리미엄, 성공 |
| Background | #FAFAFA / #FFFFFF | 깔끔함, 청결 |
| Text | #212121 / #757575 | 가독성 |

### 4.2 디자인 원칙
1. **신뢰감**: 의료기기 관련 사업이므로 전문적이고 신뢰할 수 있는 느낌
2. **따뜻함**: "내 몸을 따뜻하게" 슬로건에 맞는 온기 있는 디자인
3. **명확성**: 40~60대 타겟에 맞는 큰 글씨, 직관적 네비게이션
4. **모바일 우선**: 반응형 디자인 필수

### 4.3 UI/UX 가이드라인
- 본문 폰트: 최소 16px
- 버튼: 충분한 터치 영역 (최소 44px)
- 색상 대비: WCAG AA 기준 충족
- 로딩 속도: 3초 이내

- 창업 비용: 2,860만원
- 매장 요건: 10평 이상 (임대 또는 소유)

| 대리점 발굴 보상 | 500만원 |
| 기타 지원 | 간판, 홍보물, 인테리어 시안, 교육 시스템, 고객관리 시스템 |

1. 의료기기 판매업 신고증
2. 건강기능식품 영업 신고증
3. 건강기능식품 법정 교육 이수증
4. 임대차 계약서
